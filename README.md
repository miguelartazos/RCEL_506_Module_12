# 🚲 Ecobici CDMX Dashboard

An interactive web application built with Python and Streamlit to visualize, analyze, and track data from Mexico City's public bicycle sharing system, **Ecobici**.

## 🌟 Features
* **Real-Time Station Status:** Live tracking of available bikes and empty docks across all Mexico City stations.
* **Interactive Maps:** Visual representation of station locations and bike availability using geospatial mapping.
* **Trip Analytics:** Insights into peak usage hours, popular routes, and average trip durations.
* **Responsive UI:** A clean, easy-to-use interface powered by Streamlit that works seamlessly on both desktop and mobile.

## 🛠️ Technologies Used
* **Python 3.9+**
* **[Streamlit](https://streamlit.io/):** For the interactive web application framework.
* **Pandas:** For data manipulation and analysis.
* **PyDeck / Folium:** For interactive, high-performance mapping.
* **Requests:** For fetching real-time data from the official Ecobici API.
